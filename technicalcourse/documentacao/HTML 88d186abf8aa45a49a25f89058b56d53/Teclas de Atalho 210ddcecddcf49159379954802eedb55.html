Teclas de Atalho

Ctrl + F - Serve para pesquisar alguma palavra dentro de um site

Crtl + Espaço - Serve para pesquisar os arquivos que está dentro de sua pasta, para indicá-los de maneira mais rápida

Alt + | - Se você segurar ”alt“ + clicar com o mouse onde você quer colocar o marcador de digitação você adiciona varios marcadores, o que te permite alterar vários códigos ao mesmo tempo, isso acaba te fazendo programar de forma mais eficiente

Ctrl + Enter - Cria uma linha abaixo da que o cursor de digitação está, mesmo se seu cursor estiver dentro de um outro texto ou comando

Ctrl + Shift + P

Você deve selecionar uma parte do seu código

Usar o Ctrl + Shift + P

Isso irá abrir uma interface lá encima, lá você selecionará a opção:

“Emmet: Wrap with Abbreviation”

Após selecionar essa opção você escolhe uma tag e a digita na interface lá encima

Isso irá envolver a parte do código que você selecionou com a tag que você quiser

Serve para você construir um código mais rapidamente

Ctrl + ; - Abre o local de comentários de forma mais simples

Ctrl + Clique - Abre um arquivo

Tabulação

Para tira a tabulação use “shift + tab”

Para acrescentar a tabulação use “tab”

Tab

Pula entre os parâmetros de uma tag

Ctrl + Shift + F

Concertar tabulação automáticamente

Ctrl + C / Ctrl + V

Ao usar “Ctrl + C” em uma linha de código você a copia

Ao usar “Ctrl + V” você cola a linha

Shift + Ctrl + seta para baixo/cima

Copia a linha que o cursor está para baixo ou para cima

  • Shift + Ctrl + seta para baixo - faz a copia da linha para baixo
  • Shift + Ctrl + seta para cima - faz a copia da linha para cima

Ctrl + Enter

Cria uma linha em branco em baixa da linha que o cursor está

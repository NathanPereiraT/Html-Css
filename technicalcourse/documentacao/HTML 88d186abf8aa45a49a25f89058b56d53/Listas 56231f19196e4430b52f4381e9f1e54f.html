Listas

<ol> - Listas ordenadas

<ol type=?> - muda o tipo de listagem

I - números romanos, A - letras, 1 - números

<ol start=?> - Serve para mudar quando a lista vai começar

Exemplo: se você quiser que a lista tenha inicio na letra b, você coloca “start=2”

<ul> - Listas não ordenadas

<ul type=?> - muda o tipo de listagem

Os tipos são disc, circle, square

Estilos personalizados de <ul> com emojis

list-style-type: '\';

Basta colocar o id de um emoji ou caractere especial depois da barra ao contrário

  • Não é compatível em todos os navegadores

Para colocar um espaço após o emoji faça como no código abaixo:

list-style-type: '\2714\00A0';

O primeiro é o id de um emoji e o segundo é o id do espaço vazio

<li> - Serve para listar os itens das listas

<dl> - Lista de definição

Nelas você coloca um termo e logo em seguida da uma definição para o termo

<dt> - Coloca um termo

<dd> Define o termo

Listas Mistas - Você dentro de uma lista pode colocar outros tipos de listas

Colunas - Determina a quantidade de colunas que a lista terá

columns: 2;

Posição da lista - Por padrão vem com valor outside, ou seja para fora do bloco

Colocando o seguinte comando a lista não ultrapassa o bloco

list-style-position: inside;

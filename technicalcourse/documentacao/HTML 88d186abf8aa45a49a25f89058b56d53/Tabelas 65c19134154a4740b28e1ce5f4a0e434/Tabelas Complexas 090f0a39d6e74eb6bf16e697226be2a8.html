Tabelas Complexas

Tabelas

As tabelas complexas do html são feitas da seguinte forma:

Primeiramente se cria o parâmetro table

Dentro de table você terá outros 3 parâmetros

Esses parâmetros serão divisões dentro de table

thead - Será o cabeçalho da tabela

tbody - Será o corpo (conteúdo) da tabela

tfoot - Será o rodapé da tabela

Não é obrigatório criar todas as 3 divisões

Todas essas divisões podem receber os comandos abaixo

tr - Cria uma linha

th - Cria uma coluna de título

td - Cria uma coluna de conteúdo

th e td só podem ser usados dentro de um tr

Abaixo temos um comando específico para fazer o título de sua lista:

caption - Título/legenda para sua tabela, ocupa largura total de uma tabela

Exemplo:

<table>
        <caption>População dos Estados do Brasil</caption>
        <thead>
            <tr>
                <th>Estado</th>
                <th>População</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>São Paulo</td>
                <td>45 919 049</td>
            </tr>
            <tr>
                <td>Minas Gerais</td>
                <td>21 168 791</td>
            </tr>
            <tr>
                <td>Rio de Janeiro</td>
                <td>17 264 943</td>
            </tr>
        </tbody>
        <tfoot>
            <tr>
                <th>Total de Habitantes</th>
                <td>- - -</td>
            </tr>
        </tfoot>
    </table>

  • Dentro do caption não se pode colocar tr,th,td, apenas se pode colocar um texto, esse texto será a legenda (título) da sua tabela, irá descrever do que se trata sua tabela
  • Independente do local que você colocar o caption dentro do seu código ele sempre aparecerá encima
  • Ocupa a largura total da tabela

Semântica do título

  • Escopo: (local onde estão os dados que pertencem ao título)

Todo th é um título da sua tabela

Em uma tabela todo título recebe dados

Para melhorar a semântica de um título podemos usar os parâmetros a seguir:

Indicar escopo - Serve para indicar o escopo do título

<th scope=""></th>

Escopo na linha - Indica que o escopo do título está na mesma linha do título

<th scope="row">

Escopo na coluna - Indica que o escopo do título está na mesma coluna do título

<th scope="col">

Escopo no grupo de linhas - Indica que o escopo do título está em todas as linhas que o título ocupa

<th scope="rowgroup">

Escopo no grupo de colunas - Indica que o escopo do título está em todas colunas que o título ocupa

<th scope="colgroup">

  • Os escopos de grupo funcionam da seguinte forma: Se um título ocupa 5 linhas e você seta ele como rowgroup indicará que os dados desse título está nas 5 linhas em que ele ocupa

Mesclagem de células

Mesclagem de colunas

<td colspan="">

Fará com que a célula se expanda para as colunas do lado direito (Se mescle com as células do lado direito)

  • Dentro das aspas se coloca para quantas colunas esta célula irá se expandir
  • Se sua tabela não tiver a quantidade de colunas que você escolheu para uma célula se expandir, ela irá se expandir ao máximo até onde há colunas

Mesclagem de linhas

<td rowspan="">

Fará com que a célula se expanda para as linhas de baixo (Se mescle com as células de baixo)

  • Dentro das aspas se coloca para quantas linhas esta célula irá se expandir
  • Se sua tabela não tiver a quantidade de linhas que você escolheu para uma célula se expandir, ela irá se expandir ao máximo até onde há linhas

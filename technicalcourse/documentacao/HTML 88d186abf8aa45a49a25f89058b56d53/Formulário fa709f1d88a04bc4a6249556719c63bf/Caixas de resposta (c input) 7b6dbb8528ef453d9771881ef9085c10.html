Caixas de resposta (c/input)

Caixa de resposta

Para criar as caixas de respostas do formulário, basta digitar input e escolher algum tipo de input, confira abaixo:

<input type="text" name="" id="">
  • Acima temos um exemplo de input:text
  • Para quase todo input você deverá adicionar um “name” e um “id” para ele, esses parâmetros são importantes para identificar o input
  • Dentro de type indicamos qual o tipo do input (caixa de resposta)
  • Existem vários parâmetros que podem ser colocados dentro de input

Tipos de caixa de respostas

Ao escrever “input” no VScode aparecem diversos tipos de inputs que você pode usar no seu formulário

Alguns exemplos são: texto, data, hora, número, imagem, arquivo, e-mail, senha, telefone, cor, range

  • Existem vários inputs para telefone porém o padrão é input:tel

Largura da caixa de respostas

Ao utilizar o parâmetro “size” em um input, irá definir a largura desse input, baseada em quantas caracteres irão caber na caixa, confira abaixo:

<input type="text" size="">
  • Size = a caixa vai ter o tamanho de quantas caracteres
  • Exemplo: se o size for 10, a caixa vai ter a largura de 10 caracteres
  • O valor padrão para size é de 20
  • Se você escrever mais letras do que se cabe na caixa será criada uma rolagem lateral para ela

Texto dentro da caixa de resposta

Se você quiser adicionar um texto para aparecer dentro da caixa de respostas, enquanto o usuário não digita nada, basta usar o parâmetro “placeholder”, confira a seguir:

<input type="text" placeholder="">
  • Dentro de “placeholder” você coloca o que você deseja que esteja escrito dentro da caixa enquanto o usuário não digita nada

Preenchimento automático geral

Para configurar o preenchimento automático do formulário inteiro, basta configurar o parâmetro “autocomplete” dentro do elemento “form”, confira abaixo:

<form autocomplete="off">
        
</form>
  • off - preenchimento automático desligado
  • on - preenchimento automático ligado
  • Por padrão o preenchimento vem ligado

Preenchimento automático específico

Para indicar ao navegador qual o tipo de preenchimento que o input deve receber, você deve configurar o parâmetro “autocomplete” dentro do input, confira abaixo:

<input type="text" autocomplete="">
  • Existem diversos valores que você pode colocar em “autocomplete”
  • Você deve colocar o valor que corresponde a o que você deseja que o usuário digite dentro do input
  • Exemplo: se você estiver pedindo uma nova senha ao usuário, coloque autocomplete=”new-password”
  • Ao fazer isso, o navegador irá indicar um preenchimento que faça sentido para aquela caixa de texto em específico

Caixas de preenchimento obrigatório

Para tornar o preenchimento de uma caixa obrigatório basta usar um parâmetro dentro de input, veja abaixo:

<input type="text" required>
  • Ao colocar o parâmetro “required” dentro do input, torna obrigatório que o usuário preencha a caixa antes de enviar
  • O próprio navegador já tem um aviso para caso o usuário não preencha um campo obrigatório

Valor padrão

Existe um parâmetro que você pode colocar dentro de um input que determina um valor padrão para ele, um valor padrão que já estará escrito antes do usuário sequer digitar, confira abaixo como fazer:

<input type="text" value="">
  • O que você colocar dentro de value será o que por padrão estará escrito ao usuário acessar o formulário

Valor padrão para data

Para adicionar um valor padrão para seu input de data você deve fazer da seguinte forma:

ano-mês-dia

  • Você deve colocar na sequência acima e separar com um “-”

Exemplo:

Por padrão quero que fique 5 de abril de 2007

Para ficar por padrão a data acima deve digitar em value: 2007-04-05

Valor padrão para hora

Para adicionar um valor padrão para seu input de hora você deve fazer da seguinte forma:

hora:minuto

  • Você deve colocar na sequência acima e separar com um “:”

Exemplo:

Por padrão quero que meio dia e meia

Para ficar por padrão a hora acima deve digitar em value: 12:30

Valor padrão para input:color

Para adicionar um valor padrão para seu input de cor basta colocar dentro de value o código hexadecimal da cor, assim como da seguinte forma:

<input type="color" value="#000000">

Valor padrão para caixa de marcação

Em um formulário com caixas de marcação, para você deixar uma caixa marcada por padrão basta colocar o parâmetro checked no input referente a caixa, confira a seguir:

<input type="radio" checked>
  • Este parâmetro funciona para os inputs radio e checkbox, porém em radio apenas uma das opções podem estar checked por padrão

Valor inicial, final e padrão do input:range

Para configurar o valor do input:range ou seja o valor inicial e final, utilizamos min e max

  • Por padrão o input range vai de 0 a 100

Se eu quiser que vá de 0 a 5, eu vou ter que colocar o min=0 e o max=5, confira a seguir:

<input type="range" min="0" max="5">
  • Para configurar o valor padrão do input:range também é com a tag value, porém você deve colocar um valor que esteja dentro do que você configurou como mínimo e máximo

Mínimo e máximo de caracteres

Para alterar o mínimo e o máximo de caracteres que uma caixa deve ter basta utilizar um parâmetro dentro de input, veja abaixo:

Mínimo:

<input type="text" minlength="">
  • Dentro de minlength você determina o mínimo de caracteres que devem conter na caixa
  • Se os dados escritos na caixa não atenderem ao mínimo de caracteres a caixa deve conter, os dados não serão enviados
  • O próprio navegador já tem um aviso para caso o usuário escreva menos caracteres que o mínimo que você estipulou

Máximo:

<input type="text" maxlength="">
  • Dentro de maxlength você determina o máximo de caracteres que devem conter na caixa
  • Se os dados escritos na caixa excederem o máximo de caracteres a caixa pode conter, os dados não serão enviados
  • O próprio navegador já tem um aviso para caso o usuário escreva mais caracteres que o máximo que você estipulou

Mínimo e máximo de caracteres no input:number

Máximo:

Para o input:number ao invés de maxlength usamos apenas max

Confira abaixo:

<input type="number" max="">

Mínimo

Para o input:number ao invés de minlength usamos apenas min

Confira abaixo:

<input type="number" min="">

input:number números decimais

Por padrão não é possível digitar números com décimos dentro do input number

Para tornar possível você deverá alterar a espera entre os números

Por padrão a espera é de 1, então, os valores aceitos aumentam de 1 em 1, ou seja, só aceita números inteiros

Veja abaixo com alterar isso:

<input type="number" step="0.1">

Ao fazer isso os valores aceitos vão aumentando em 0.1, assim fica possível adicionar números decimais

Caixas de marcação múltipla

Para criar um formulário com caixas de marcação, e com o usuário podendo selecionar várias caixas, você deverá usar a tag input:checkbox, confira a seguir:

<input type="checkbox">
  • Para esse tipo de input o mais comum é colocar label após o input, porém você pode colocar na ordem que desejar contanto que ligue o “id” com “for”
  • Exemplo de código a seguir:
<form action="cadastro.php" method="get" autocomplete="on">
        <fieldset>
            <legend>Esporte Favorito</legend>
            <br>
            <div>
                <input type="checkbox" name="bas" id="bas">
                <label for="bas"> Basquete</label>
            </div>
            <br>
            <div>
                <input type="checkbox" name="fut" id="fut">
                <label for="fut"> Futebol</label>
            </div>
            <br>
            <div>
                <input type="checkbox" name="vol" id="vol">
                <label for="vol"> Vôlei</label>
            </div>
            <br>
        </fieldset>
    </form>
  • Resultado visual:
  • O usuário pode marcar diversas caixas

Caixas de marcação única

Para criar um formulário com caixas de marcação, e com o usuário podendo selecionar apenas uma das caixas, você deverá usar a tag input:radio, confira a seguir:

<input type="radio">
  • Para esse tipo de input o mais comum é colocar label após o input, porém você pode colocar na ordem que desejar contanto que ligue o “id” com “for”
  • Os inputs radio deverão ter o mesmo “name” se não tiver não funcionará corretamente
  • Exemplo de código a seguir:
<form action="cadastro.php" method="get" autocomplete="on">
        <fieldset>
            <legend>Sexo</legend>
            <br>
            <div>
                <input type="radio" name="sex" id="fem">
                <label for="fem">Feminino</label>
            </div>
            <br>
            <div>
                <input type="radio" name="sex" id="mas">
                <label for="mas">Masculino</label>
            </div>
            <br>
        </fieldset>
    </form>
  • Resultado visual:
  • O usuário só vai poder marcar uma única caixa, se ele tentar marcar duas a outra irá se desmarcar

Caixa de envio

Todo formulário deve ter um botão para que o usuário possa mandar as respostas

Para criar esse botão você também irá usar um input

O input para criar essa caixa se chama input:submit

<input type="submit" value="">
  • Dentro de value você coloca o que você deseja que esteja escrito dentro da caixa

Caixa de limpeza

Alguns formulários tem um botão para limpar o que o usuário escreveu dentro das caixas de texto

Para criar esse botão você também irá usar um input

O input para criar essa caixa se chama input:reset

<input type="reset" value="">
  • Dentro de value você coloca o que você deseja que esteja escrito dentro da caixa

Propriedades do imput file

  • Ao usar o imput file, você deverá usar o método post, pois se você não usar o arquivo que o usuário selecionou não será enviado
  • No imput file você também deverá usar o parâmetro enctype="multipart/form-data", da seguinte forma:
<input type="file" name="arquivo" id="arquivo" enctype="multipart/form-data">

Programação para quando o usuário inserir dados na caixa de resposta

Você pode usar um parâmetro nas suas caixas de respostas que tem o nome de “oninput”, este parâmetro ativa uma função toda vez que o usuário adicionar ou trocar um dado dentro do input, veja a seguir:

<input type="number" oninput="">
  • Toda vez que o usuário colocar ou trocar um dado vai ativar a função javascript que estiver dentro do parâmetro “oninput”

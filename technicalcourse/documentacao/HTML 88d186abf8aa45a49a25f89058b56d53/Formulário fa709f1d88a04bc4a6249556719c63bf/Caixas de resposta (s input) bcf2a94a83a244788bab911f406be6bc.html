Caixas de resposta (s/input)

Caixa com opções de resposta pré-selecionadas

Você pode criar uma caixa de resposta, cujo as opções de resposta do usuário sejam pré-selecionadas por você, para isso usamos a tag “select”, confira abaixo:

<select name="est" id="iest">
                
</select>
  • Dentro da tag “select” você deverá colocar as opções que o usuário poderá escolher
  • Para colocar as opções de resposta, utiliza-se a tag “option”, confira abaixo:
<select name="est" id="iest">
    <option value=""></option>
</select>
  • Entre a tag de abertura e fechamento você deverá escrever uma opção de resposta para o usuário
  • Dentro do parâmetro “value” coloque um código para identificar o que está escrito na opção de resposta

Grupo de opções de resposta

Você pode dividir as opções de resposta em grupos, cada grupo de resposta deve ficar na tag “optgroup”, confira a seguir:

  • Use o parâmetro “label” em “optgroup” para definir um nome ao grupo de opções
<select name="est" id="iest">

                <optgroup label="Região Sudeste">
                    <option value="MG">Minas Gerais</option>
                    <option value="RJ">Rio de Janeiro</option>
                    <option value="SP">São Paulo</option>
                    <option value="ES">Espírito Santo</option>
                </optgroup>

                <optgroup label="Região Sul">
                    <option value="RS">Rio Grande do Sul</option>
                    <option value="SC">Santa Catarina</option>
                    <option value="PR">Paraná</option>
                </optgroup>
                
</select>
  • Acima podemos ver que dividimos as opções de resposta em dois grupos, assim deixando as opções de resposta mais organizadas

Opção de resposta padrão

Para definir uma opção de resposta como padrão, basta adicionar dentro da tag “option” o parâmetro “selected”, assim essa opção de resposta ficará selecionada por padrão, veja a seguir:

<option value="" selected></option>

Opção de resposta vazia

É recomendado deixar a opção de resposta padrão vazia, para que não fique nenhuma opção já pré-selecionada para o usuário, veja abaixo como fazer:

<option value="" selected> --- </option>
  • Não colocamos a opção em nenhum grupo de opções de resposta
  • Não escrevemos nada no parâmetro “value”
  • Dentro da tag option não digitamos nada, ou então, digitamos algo pedindo para que o usuário selecione uma resposta
  • E por fim adicionamos o parâmetro “selected” para esta opção de resposta ficar selecionada por padrão

Caixa de resposta com respostas pré-selecionadas

  • Você utiliza isso apenas em input:text

Você pode colocar algumas respostas pré-selecionadas dentro da caixa de resposta, para isso usamos a tag “datalist”, confira abaixo:

  • Para abrir a tag “datalist” basta digitar “datal”
<datalist>
                
</datalist>
  • Dentro da tag “datalist” você pode colocar opções de resposta para o usuário por meio da tag “option”, assim como na tag “select”
  • As tags “option” dentro de “datalist” não precisam do parâmetro “value”, então pode apagar da tag ou deixar vazio

Para na caixa de resposta as opções de respostas adicionadas na tag “datalist” ficarem disponíveis para o usuário escolher, você deverá linkar “datalist” com o input:text referente a caixa de resposta, para fazer isso você deverá dar um id para “datalist” e colocar esse id dentro do parâmetro list no input:text, veja abaixo:

<input type="text" list="prof">
            <datalist id="prof">
                
            </datalist>
  • Ao fazer isso todas as opções de resposta que você adicionar em datalist, ficarão disponíveis para o usuário as usarem, isso irá facilitar o usuário a preencher o formulário

Caixa de texto

Em um formulário você pode criar uma caixa de texto, e nela o usuário poderá digitar um texto, para fazer isso você usa a tag “textarea” que funciona de forma parecida com um input, confira a seguir:

<textarea name="" id="" cols="" rows=""></textarea>
  • “name” e “id” terão a mesma funcionalidade que tem em qualquer outro input
  • “id” também será usado para linkar a tag “textarea” com a tag “label”
  • cols - serve para definir o número de colunas da área, por padrão vem 30
  • rows - serve para definir o número de linhas da área, por padrão vem 10

Código de exemplo mostrando o uso de textarea:

<label for="itex">Texto: </label>
<textarea name="tex" id="itex" cols="30" rows="10"></textarea>
  • O que você digitar entre a tag de abertura e fechamento de “textarea” vai ficar escrito dentro da caixa quando o usuário acessar (não é recomendado que você faça isso)
  • Em formulários que usam a tag “textarea” você deve usar o método post, para evitar que sejam ultrapassados os 3000 bytes que o método get suporta

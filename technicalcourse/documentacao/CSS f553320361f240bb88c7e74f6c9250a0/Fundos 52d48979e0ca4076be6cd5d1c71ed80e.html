Fundos

  • Mesmo que você utilize uma imagem de fundo, é importante você configurar uma cor de fundo, pois seu site pode demorar um pouco para carregar, e enquanto não carregar mostrará esta cor de fundo configurada

Cor de fundo

O parâmetro background-color é usado para determinar uma cor de fundo a um elemento

background-color: #000000;

Cor de fundo em degradê

Direção do degradê

Você pode alterar a imagem de fundo, acrescentando um degradê, basta colocar a direção que você quer (Em quantos graus, 90 graus = 90deg, se você colocar um valor negativo no deg vai inverter a direção do degradê), e indicar em quais cores seu degradê irá passar

background-image: linear-gradient(90deg, yellow, red, purple);

Direção do Degradê (Forma Simples) - Basta colocar no lugar do parâmetro ”deg“ o parâmetro ”to (a direção que você quer)“

Direções: top (cima), right (direita), bottom (baixo), left (esquerda)

background-image: linear-gradient(to bottom);

Degradê Fixo ao Fundo

<style>
        body
        {
            background-attachment: fixed;
        }
</style>

Degradê Circular

<style>
        body
        {
            background-image: radial-gradient(circle, yellow, red, purple);
        }
</style>

Volume de Cores - Ao colocar uma porcentagem após cada cor, muda o volume da cor dentro do degradê

<style>
        body
        {
            background-image: linear-gradient(45deg, yellow 2%, red, purple 50%);
        }
</style>

Imagem de Fundo

Carregar imagem para ser o fundo de um elemento

Para carregar uma imagem de fundo a um elemento primeiramente adicione o parâmetro background-image dentro do estilo desse elemento, depois deverá adicionar o parâmetro url dentro do parâmetro background-image

A partir daí temos duas formas de selecionar o arquivo que será fundo de nosso elemento:

Imagem local, imagem que está armazenada junto ao arquivo de seu documento, para adicionar basta escrever o nome do local do arquivo

background-image: url(pattern001.png);

Imagem externa, basta adicionar o link de uma imgem, porém se essa imagem sair do ar ela saíra também de seu site

background-image: url(https://c4.wallpaperflare.com/wallpaper/500/442/354/outrun-vaporwave-hd-wallpaper-preview.jpg);

Configurações de fundo

Tamanho da imagem de fundo

Você pode configurar qual será o tamanho da imagem de fundo de um elemento

  • A primeira definição de tamaho será a largura da imagem e a segunda a altura
background-size: 100px 400px;
  • Se você colocar o tamanho do fundo de 100% ele irá ocupar por completo o fundo do elemento no qual ele é fundo
background-size: 100% 100%;

Existem outros 2 valores para configurar o tamanho do fundo, esses valores são:

background-size: contain;
  • A imagem de fundo aparecerá por completo, porém isso pode ocasionar em a sua imagem de fundo não cobrir o fundo por completo, pois a este parâmetro se preocupa em apenas mostrar toda a imagem de fundo
background-size: cover;
  • A imagem de fundo cobre todo o elemento a qual ela pertence, porém isso pode ocasionar em não mostrar a imagem de fundo por completo, cortando algumas partes dela para preencher o fundo por completo

Posição do fundo

A posição do fundo é determinada em relação ao elemento do qual o fundo pertence

  • Se a imagem do fundo for maior do que o elemento do qual o fundo pertence, irá abrir apenas abrir uma parte da imagem, que seja relativa ao tamanho do elemento a qual ela está. O parâmetro background-position irá definir qual parte dessa imagem irá abrir. Exemplo: se o background-position for “bottom left” irá abrir a imagem a partir do ponto inferior esquerdo até o tamanho que o elemento tem
  • Se a imagem for menor do que o elemento onde ela se encontra a única coisa que irá acontecer é o fundo ficar posicionado no espaço que você designou

Posições:

Para designar a posição da sua imagem você deve dizer duas posições

Vertical: esquerda, centro, direita

Horizontal: cima, centro, baixo

  • Você deve escolher apenas um valor vertical e um horizontal
  • Se você escolher baixo e direita, a imagem irá se posicionar no canto inferior direito

background-position: center center;

Repetição da imagem de fundo

  • Se a imagem que você designou não ocupar o fundo de um elemento por completo, esta imagem irá repetir até preencher todo o fundo do elemento, isso é por padrão

Para a imagem de fundo repetir até ocupar o elemento por completo (configuração padrão)

background-repeat: repeat;

Para a imagem de fundo não repetir

background-repeat: no-repeat;

Para a imagem de fundo só repetir horizontalmente

background-repeat: repeat-x;

Para a imagem de fundo só repetir verticalmente

background-repeat: repeat-y;

Tipos de fundo

background-attachment: fixed;

A imagem fica fixa ao fundo

  • Este tipo de fundo também é usado para criar o efeito parallax

background-attachment: scroll;

A imagem não fica fixa ao fundo, ou seja, ao mover o site para baixo o fundo não acompanha

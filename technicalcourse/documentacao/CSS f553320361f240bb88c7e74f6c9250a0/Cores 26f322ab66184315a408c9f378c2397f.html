Implantar Cores

  • Nome

Você coloca o nome da cor em inglês para designar uma cor, essa técnica é fácil de ser aplicada porém temos um limite nas cores que podem ser usadas

color: black

  • Códigos Hexadecimais

(Hexadecimal é uma base numérica onde os números são representados de 0 até a letra F ou seja 16 algarismos)

Cada cor tem um código hexadecimal para representá-la

(Os dois primeiros dígitos representa uma variação de vermelho, os dois segundos de verde e os dois últimos de azul)

"color: #000000;"

  • RGB

Representar cores a partir da variação vermelho, verde e azul, indo de 0 a 255

color: rgb(0, 0, 0)

(A primeira casa representa o vermelho, a segunda o verde, a terceira o azul

  • HSL

Representar a partir de matiz, saturação e luminosidade

Matiz vai de 0 a 359

Saturação e luminosidade de 0% a 100%

color: hsl(0, 0%, 0%)

  • Pelo VScode

Outro jeito de designar uma cor é pelo Visual Studio Code, basta selecionar uma cor aleatória e depois passar o mouse por cima do nome dela, assim abrirá uma tela onde você pode escolher uma cor diretamente da paleta

Você até pode designar de qual forma ficará no código, RGB, HSL ou Hexadecimal

Degradê

Você pode alterar a imagem de fundo, acrescentando um degradê, basta colocar a direção que você quer (Em quantos graus, 90 graus = 90deg, se você colocar um valor negativo no deg vai inverter a direção do degradê), e indicar em quais cores seu degradê irá passar

background-image: linear-gradient(90deg, yellow, red, purple);

Degradê Fixo ao Fundo

<style>
        body
        {
            background-attachment: fixed;
        }
</style>

Degradê Circular

<style>
        body
        {
            background-image: radial-gradient(circle, yellow, red, purple);
        }
</style>

Volume de Cores - Ao colocar uma porcentagem após cada cor, muda o volume da cor dentro do degradê

<style>
        body
        {
            background-image: linear-gradient(45deg, yellow 2%, red, purple 50%);
        }
</style>

Direção do Degradê (Forma Simples) - Basta colocar no lugar do parâmetro ”deg“ o parâmetro ”to (a direção que você quer)“

background-image: linear-gradient(to bottom);

Media Queries

Conceito de Media Query

As media queries servem basicamente para você poder moldar seu site para os mais diversos tipos de tamanhos de tela

Você basicamente configura para o mesmo documento HTML vários estilos CSS diferentes, cada uma para um tamanho de tela

Especificando uma página CSS

Existe um parâmetro que você coloca dentro do “link:css”, este parâmetro indica para qual mídia a página CSS será destinada

O parâmetro que serve para indicar isso é o parâmetro “media”, confira a seguir:

<link rel="stylesheet" href="style.css" media="">

Valores que podem ser colocados dentro de “media”:

all - Significa que a página CSS é para tudo

screen - Significa que a página CSS é para uma tela

print - Significa que a página CSS é para uma impressora

  • Se você for colocar dois parâmetros dentro de media você deve separá-los por “and”
  • Na ordem dos links “all” sempre deve ser o último

Orientação de media

Existem dois tipos principais de exibição para o conteúdo de um site, confirá-os abaixo:

portrait - Altura maior que largura (celular em pé)

landscape - Largura maior que altura (celular deitado)

  • Se você for colocar dois parâmetros dentro de media você deve separá-los por “and”
  • As orientações de media devem ser escritas cada uma dentro de um parênteses, isoladamente
  • Dentro dos parênteses você especifica a orientação pelo parâmetro orientation, confira a seguir:
<link rel="stylesheet" href="style.css" media="screen and (orientation: landscape)">
<link rel="stylesheet" href="style.css" media="screen and (orientation: portrait)">

Exemplo de media query

No exemplo abaixo mostramos o uso de media query:

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Media Query</title>
    <link rel="stylesheet" href="style.css" media="screen">
    <link rel="stylesheet" href="impressora.css" media="print">
</head>
  • Dentro de head linkamos dois estilos CSS, um para tela e outro para impressora
  • Iremos configurar o estilo de cada página CSS de forma diferente, isso irá gerar duas media queries diferentes
  • Uma dessas media queries será exibida normalmente na tela do usuário quando ele acessar o site, e outra será para se o usuário imprimir o site

Media query para impressora

Você deve criar uma media query específica para quando um usuário for imprimir sua página, pois tem alguns elementos do seu site que não são necessários que saiam na impressão ou então que saiam de forma diferente, por isso devemos configurar uma media query específica para caso alguém queira imprimir seu site

Criar diversas mídias em um único CSS

Dentro de um único CSS você pode criar diversas mídias, para fazer isso você deve utilizar a regra “@media” (você pode abrir a estrutura dela digitando “@media screen”), confira abaixo:

@media screen {
            
        }
  • No lugar de screen você pode colocar outro tipo de mídia
  • As configurações da regra media são feitas da mesma forma do parâmetro media, que fica no link:css, confira abaixo:
@media screen and (orientation: portrait){
            
        }
  • Se você for colocar dois parâmetros dentro dessa regra você deve separá-los por “and”
  • As orientações (orientation: ) devem ser escritas cada uma dentro de um parênteses, isoladamente
  • Dentro dos parênteses você especifica a orientação, pelo parâmetro orientation
  • Dentro das chaves você configura CSS normalmente

Configurando um estilo para cada tamanho de tela

  • Pesquise por device breakpoint para saber o tamanho de tela para cada tipo de dispositivo

Você pode configurar um estilo para cada tamanho de tela, ou por meio de um parâmetro “media” dentro de um link:css, ou então, dentro da regra media

  • Você deve defini-los dentro de um parênteses

Confira abaixo um exemplo:

@media screen and (min-width: 768px) and (max-widht: 992px) {
    
}
  • Se a tela do usuário possuir entre 768 e 992 pixels o estilo acima será utilizado
  • Dentro das chaves você configura CSS normalmente

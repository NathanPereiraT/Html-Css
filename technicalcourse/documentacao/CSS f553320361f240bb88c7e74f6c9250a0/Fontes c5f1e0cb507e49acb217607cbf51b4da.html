@font-face - Permite você implantar uma fonte baixada

Ao digitar essa regra abre a seguinte estrutura:

@font-face 
        {
            font-family: ;
            src: url();
        }
  • src - Aqui devemos indicar o nome do arquivo onde está a fonte
  • font-family - Aqui devemos dar um nome para nossa fonte

(Para tudo acima funcionar devemos acrescentar um parâmetro após scr, este parâmetro se chama format, abaixo vemos como deve ser feito)

@font-face 
        {
            font-family: ;
            src: url() format();
        }

Tipos de formatos:

  • opentype
  • truetype
  • embedded-opentype
  • truetype-aat
  • svg

(Cada navegador suporta melhor um tipo de fonte então em um site e bom ter diversos tipos)

(Os dois principais tipos são opentype e truetype)

@font-face
        {
            font-family: 'medieval';
            src: url(OldLondon.ttf) format('truetype');
        }

(Toda vez que precisar usar esta fonte você deve se referir a ela usando o nome que você deu a ela em font-family)

Compatibilidade de Fontes

Existem dispositivos que não suportam certos tipos de fontes, por esse motivo sempre é bom usar uma “safe combination”, uma combinação de fontes parecidas que ajudam a evitar que ocorra o problema do dispositivo não suportar a fonte

body
        {
            font-family: Arial, Helvetica, sans-serif;
        }
  • Primeiro o dispositivo vai tentar carregar a “Arial”, se não conseguir vai tentar a Helvetica, e assim por diante (Seguindo a sequência)s

Sempre que você colocar serif, sans-serif, cursive ou monoespace ele vai carregar uma fonte aleatória da categoria, pois todos os dispositivos tem pelo menos uma fonte dessas categorias

Tamanho da Fonte

font-size - Modifica o tamanho da fonte

Podemos ver diversas outras opções apertando “ctrl” + “espaço” mas “em” é a forma mais recomendada de se delimitar tamanho

  • em - Muda o tamanho de acordo com o selecionado primeiramente, se você selecionou 100px e usar “0.5em” logo o tamanho será 50px

h1
        {
            font-size: 1em;
        }
  • A recomendação é de sempre que for selecionar o tamanho utilize ou em ou px

Largura da Fonte

Largura ou peso é o nome que se da a largura de uma fonte

Um importante aviso é dizer que nem toda fonte tem variações de largura, na verdade a maioria não tem

Exemplos de variações:

Lighter = Um pouco mais fina que a normal

Normal = A largura normal

Bold = Um pouco mais grossa que a normal

Bolder = Um pouco mais grossa que a bold

100 a 900 = Outro jeito de selecionar a variação de largura 

h1
        {
            font-weight: 300;
        }
  • Nem toda fonte possui variação de largura

Itálico

h1
        {
            font-style: italic;
        }
  • Nem toda fonte possui itálico

Sublinhado

h1
        {
            text-decoration: underline;
        }
  • Nem toda fonte possui sublinhado
  • Existem outros parâmetros utilizáveis em ”text-decoration“
  • Ao usar “text-decoration” “none” o texto fica sem sublinhado

O exemplo abaixo mostra como mudar a cor do sublinhado (Se não definir uma cor ficará da cor do elemento que foi sublinhado)

text-decoration-color: black;

Variações

É um parâmetro que te permite fazer alterações na sua fonte

font-variant: small-caps;

Existem vários valores que podem ser colocados em font-variant, e cada um deles dá uma disposição diferente à sua fonte

  • small-caps - faz a parte em que a fonte foi setada como small-caps ficar com design de letra maiúscula porém com tamanho de letra minúscula

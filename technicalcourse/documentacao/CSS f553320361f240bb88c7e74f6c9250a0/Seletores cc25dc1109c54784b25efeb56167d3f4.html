Tradução HTML x CSS

“id” em HTML e “#” em CSS

“class” em HTML e “.” em CSS

“pseudo-class” em HTML e “:” em CSS, existem diversas pseudo classes

“pseudo-element” em HTML e “::” em CSS, existem diversos pseudo elementos

“children” em HTML e “>” em CSS

  • Um elemento pode ter id e class, porém as configurações de id sobrepõem as de class

  • Conteúdos (content) - Em CSS conteúdos são indicados como “content”

  • Quando você for colocar estilo em algum seletor criado, o nome antes dele não muda nada

id

Ao definir um id a um elemento, você torna especial esse tal elemento, o diferenciando dos da mesma

classe

Com isso quando você for adicionar estilo ao seu site, você poderá fazer um estilo personalizado para este elemento em específico

Abaixo mostra como definir um id a um elemento

<h1 id="prime"></h1>

No exemplo acima nós tornamos este “h1” diferente dos outros dando uma especificação diferente a ele

Abaixo mostramos como fazer um estilo para este id que foi criado

h1#prime
{
    
}

O elemento com esse id seguirá o estilo que você deu ao id

  • O exemplo acima mostra a forma que devemos referir-se ao “h1” especial quando formos dar estilo ao CSS. É com o nome que damos ao id dele, mas antes uma hashtag que dentro das CSS indica id
  • Não se pode usar o mesmo id em dois elementos diferentes (Apenas se for em páginas diferentes)

Class

Ao indicar que um elemento faz parte de uma class ele começa a seguir o estilo dessa tal classe e não igual ao dos outros elementos dessa classe

<h1 class="prime"></h1>

Abaixo mostramos como fazer um estilo para esta classe

h1.prime
{

}

  • A única diferença de id para class e que id só serve para alterar um único elemento, já class serve para você alterar um grupo de elementos que foram indicados como pertencentes a uma classe em específico

  • Tem como indicar que um elemento faz parte de duas classes diferentes, como no exemplo abaixo abaixo:
<h1 class="prime alpha"></h1>

Pseudo Classes

Existem diversas pseudo classes, elas determinam como o estilo de algum elemento dependendo do estado em que ele se encontra

div:hover{
            background-color: blueviolet;
        }

Neste exemplo acima, estamos utilizando o parâmetro hover (Parâmetro ativado quando seu mouse está encima de algum elemento)

Isto acima indica que quando o mouse for passado encima da “div” está “div” mudará de estilo de acordo como você configurou para esta pseudo classe

  • Você pode colocar um tempo para a transição entre o estilo normal e o depois da psudo classe ser ativada, esse tempo pode ser indicado em segundos (s) ou milisegundos (ms)
div{
    transition-duration: 0.5s;
}

Essa tag deve ser usada na área do estilo antes da pseudo classe ser ativada

Tipos de Pseudo Classes

hover - Quando o mouse estiver em cima do elemento o estilo se altera

visited - Quando o link do elemento já foi visitado o estilo se altera

active - Quando clicarem encima do elemento o estilo se altera

focus-within - Quando uma caixa está selecionada o estilo se altera

nth-child() - Serve para alternar os estilos de um mesmo elemento (usado prinicipalmente para fazer o efeito zebrado), exemplo: td:nth-child(2n), fará com que a cada 2 dados da tabela um fique com o estilo diferente

Elemento Filho

Você pode estilizar um elemento que está dentro de outro de uma forma específica

Por exemplo, se eu tiver parágrafos dentro de uma div, e eu quiser dar um estilo específico a esses parágrafos dentro da div eu posso usar o código abaixo:

div > p{
            
        }

Com o código acima eu indico que este estilo só irá afetar os parágrafos que estejam um nível dentro de alguma div, já com o código abaixo todos os parágrafos que estiverem em uma div independente do nível serão afetados pelo estilo

div p{
    
}
  • Ao colocar (div > p) você indica que só serão afetados pelo estilo os elementos que estiverem a um nível de distância de div, ou seja os parágrafos que estiverem dentro de um aside, que por sua vez estão dentro de uma div, não serão afetados
  • Ao colocar (div p) você indica que independente da distância que o parágrafo estiver de div ele será afetado

Um exemplo abaixo:

div > p{
            display: none;
        }

        div:hover > p{
            display: block;

        }

Neste exemplo acima nós ocultamos todos os parágrafos dentro de div, mas quando passarmos o mouse encima da div, este parágrafo irá se revelar

Pseudo elemento

Serve para configurar/adicionar elementos dentro de outros elementos

a::after{
            content: '';
        }

No exemplo acima adicionaremos um pseudo elemento ao elemento “a”

  • Existem diversos tipos de pseudo-elementos que podemos adicionar aos elementos

Tipos de Pseudo Elementos

after - Serve para implementar algo após um elemento

before - Serve para implementar algo antes de um elemento

webkit-scrollbar - Serve para estilizar a barra de rolagem

Selecionar elementos pelo valor de sua variável

Você pode dar um estilo a um elemento o espececíficando por meio da valor da váriável dele, veja um exemplo abaixo:

input[type="submit"] {
    display: block;
}
  • Para fazer você deve digitar o nome de um elemento e após ele entre parênteses você especifica um parâmetro dele
  • No exemplo acima nós configuramos o estilo apenas dos inputs que são do tipo submit

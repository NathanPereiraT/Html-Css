Transições

  • Todas transições devem vir por último em uma configuração de estilo

Explicando a ativação de uma transição

  • O exemplo abaixo usa a ativação de pseudo-classes para explicar como a transição de estilo acontece, porém uma transição é ativa sempre que um elemento muda de estilo, independentemente se a causa para troca de estilo foi uma pseudo-classe

Você pode fazer com que um elemento troque de estilo quando uma pseudo-classe for ativa

E você pode definir um tempo para troca do estilo quando a pseudo-classe for ativada

Esse tempo tempo para troca de estilo, gera uma espécie de animação

Veja abaixo como definir o tempo para que a troca de estilo aconteça

section#redes-sociais img:hover {
    border: 2px solid rgba(255, 255, 255, 0.6);
    transform: translate(-3px, -3px);
    box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
}

Quando a pseudo classe hover for ativada o estilo do elemento vai passar a ser o estilo acima

Por não termos definido um tempo para que essa troca de estilo aconteça, assim que a pseudo-classe for ativada o estilo mudará

Tempo de transição específico para cada estilo

Abaixo mostramos como definir um tempo para troca de estilo:

section#redes-sociais img:hover {
    transition: transform 0.1s, border 0.5;
}

Dentro de transition você deve colocar o nome de um parâmetro e depois do nome o tempo que levará para a troca do estilo acontecer

Você deve separar o nome do parâmetro e o tempo de troca dele por uma vírgula

No exemplo acima definimos que a troca de estilos do transform durará 0.1 segundo, a troca de border 0.5 segundos e a troca do box-shadow não definimos então ocorrerá sem delay

Tempo de transição para todos os estilos

Se você quiser configurar um tempo para que todos os estilos mudem juntos use o parâmetro abaixo:

section#redes-sociais img:hover {
		transition-duration: 1s;
}

No exemplo acima todos os estilos mudarão em um segundo

Configurar animação da transição

Para configurar a animção de transição utilizamos o parâmetro transition-timing-function, dentro desse parâmetro temos valores já prontos que você pode utilizar (cada um desses valores muda a forma como a animação acontece), ou então você pode configurar manualmente como a animação irá ocorrer, confira abaixo os exemplos:

  • Valores prontos
div#um {
    transition-timing-function: ease;
}
  • Configuração manual
div#um {
    transition-timing-function: cubic-bezier(0.075, 0.82, 0.165, 1);
}

cubic-bezier() é o valor para configurar a animação manualmente

Ela define quanto tempo cada passo da animação irá durar, cada passo pode váriar de 0 a 1, na escala de quanto tempo vai durar

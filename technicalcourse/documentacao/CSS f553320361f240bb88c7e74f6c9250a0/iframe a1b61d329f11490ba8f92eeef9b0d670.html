iframe

São caixas que exibem dentro do seu site algum outro site

Você define o site que será aberto a partir do link que você adiciona no iframe

  • iframes são recomendados apenas para situações específicas
  • O iframe é um bloco de display inline
  • O tamanho padrão de um iframe é de: 300px de largura e 150px de altura

Link do iframe

Para linkar um site em seu iframe escreva a url do site no parâmetro src do iframe

  • Alguns sites não permitem que você crie iframe deles
<iframe src="https://www.cursoemvideo.com">

</iframe>
  • Você pode linkar no seu iframe uma página local do seu documento, para isso basta indicar a página dentro de src, assim como no exemplo abaixo:
<iframe src="teste.html">

</iframe>

Borda do iframe

A borda do iframe é definida a partir do parâmetro frameborder, este parâmetro é bem limitado pois ele só aceita os valores 0 e 1, ou seja, ou sem borda ou com a borda de 1 pixel

  • Vem por padrão setado como zero, ou seja, sem borda
  • É recomendado usar CSS para configurar a borda de um iframe ao invés do parâmetro frameborder, pois frameborder é um parâmetro muito limitado
<iframe src="https://www.cursoemvideo.com" frameborder="0">

</iframe>
  • Para configurar a borda de um iframe com CSS, é da mesma forma que configuramos a borda de qualquer outro elemento

iframe inacessível

Alguns aparelhos ou navegadores não são compatíveis com iframe

Você pode colocar uma mensagem no lugar em que apareceria o iframe para indicar ao usuário que o iframe não pode ser aberto

Tudo oque você acrescenta entre a tag de abertura e fechamento de um iframe é o que será exibido caso o aparelho não consiga exibir o iframe

<iframe src="https://www.cursoemvideo.com" frameborder="0">
    <p>iframe não pode ser exibido</p>
    <a href="https://www.cursoemvideo.com">Clique aqui para acessar</a>
</iframe>
  • É recomendado colocar o link do site que seria aberto no iframe, caso o iframe não possa ser acessado

Tamanho do iframe

Para alterar o tamanho de um iframe é da mesma forma que se altera qualquer outro elemento, basta configurar o style do iframe dentro de uma CSS

Scroll no iframe

Se o conteúdo de um site for maior do que seu iframe, o iframe irá gerar uma barra de rolagem, o parâmetro scrolling define se haverá ou não haverá barra de rolagem no iframe

  • scrolling=”auto” - cria barra de rolagem se o conteúdo do iframe exceder o tamanho dele (parâmetro padrão)
  • scrolling=”no” - sem barra de rolagem
  • scrolling=”yes” - com barra de rolagem
<iframe scrolling="no">

</iframe>
  • Sem a barra de rolagem o iframe só irá exibir o início do site, no caso a parte que couber no iframe

Abrir os links do site no iframe

Você pode fazer com que os links abertos sejam exibidos dentro de um iframe, para isso siga os passos a seguir

Primeiramente você deverá nomear o iframe que você deseja que link seja aberto, para isso use o parâmetro name, assim como no exemplo abaixo:

<iframe name="frame1" src="">
        
</iframe>
  • Neste caso podemos deixar src vazio, pois só é para o iframe exibir a página quando o link for aberto

Depois de nomear o iframe, você terá que indicar o nome dele dentro do parâmetro target de um link

Ao fazer isto este link ao ser acessado será aberto dentro do iframe que foi indicado no parâmetro target

<a href="#" target="frame1">Página 1</a>

Conteúdo do iframe por HTML

Você pode colocar no seu iframe o conteúdo por meio de HTML, assim como a seguir:

<iframe srcdoc="<h1>Abra algum dos links acima</h1>">

</iframe>

Basta adicionar o parâmetro srcdoc no iframe, e dentro de srcdoc escrever parâmetros em HTML, estes parâmetros serão exibidos dentro do iframe

  • É recomendado você ao invés de usar srcdoc para exibir conteúdos no iframe, você criar outro documento e linkar ele no iframe

Problemas no iframe

  • Segurança

iframes podem apresentar falhas de segurança, pois o iframe é uma porta que você abre para outro site dentro do seu, ao abir essa porta alguns sites maliciosos podem agir de forma errada no seu site, podendo capturar e rastrear dados do usuário e do seu site, devido a isso você deve tomar muito cuidado para qual site você abrirá a porta, este site tem que ter segurança e sua confiança.

Apesar dos iframes não serem muito seguros, você pode tomar algumas medidas para aumentar a segurança deles

  • Incompatibilidade

Nem todos os aparelhos ou navegadores são compatíveis com iframe, e além disso existem alguns sites que não permitem que você crie um iframe deles

  • Mecanismos de busca

Os mecanismos de busca do google apresentam alguns problemas para indexar conteúdos que estão dentro de iframes, ou seja, não utilize muitos iframes em seu site e se for usar não coloque nele um conteúdo que seja muito importante para seu site

  • Acessibilidade

Isso pode prejudicar a acessibilidade do seu site, tendo em vista que programas de leitores de tela para pessoas cegas apresentam falhas para ler conteúdos que estão dentro de iframes

  • Responsividade

Se o site que linkou em um iframe não for responsivo, pode acontecer do conteúdo ficar quebrado e desorganizado dentro do iframe

  • Navegação

Alguns erros podem acontecer ao usuário navegar no seu iframe, a página pode não voltar ou alguma página pode não abrir, esses erros acontecem ou por conta do próprio site que foi linkado no seu iframe ou por conta do navegador

Segurança

Iframes não são muito seguros, porém você pode tomar algumas medidas para torná-los mais seguros

Para torná-los mais seguros utilizamos dois parâmetros:

  • Impedir as ações do iframe

Há um parâmetro que te permite reduzir as ações que o iframe pode realizar, este parâmetro se chama sandbox, ele está sendo representado a seguir:

<iframe src="frames/frame03.html" sandbox="">

</iframe>

Dentro dele você pode colocar vários valores, cada valor diminui algum tipo de ação que o iframe pode realizar, veja os valores a seguir:

sandbox - Não permite nenhum tipo de ação

allow-same-origin - Permite que apenas iframes de sites que se localizam no mesmo documento (sites locais) tomem ações

allow-forms - Permite o uso de formulários

allow-scripts -Permite o uso de scripts

Você pode colocar dentro de sandbox vários dos valores acima

Tem vários outros valores que podem ser colocados em sandbox, para vê-los basta colocar o cursor entre as aspas e pressionar crtl+espaço

  • Impedir a coleta de dados do iframe

Para impedir a coleta de dados utilizamos o parâmetro referrerpolicy, assim como a seguir:

<iframe src="frames/frame03.html" referrerpolicy="">
        
</iframe>

Há dois valores que podem ser colocados nele:

no-referrer-when-downgrade - Valor padrão, que permite a coleta de dados

no-referrer - Não permite a coleta de dados

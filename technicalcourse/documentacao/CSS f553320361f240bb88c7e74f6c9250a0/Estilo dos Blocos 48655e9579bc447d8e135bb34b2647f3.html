Estilo dos Blocos

Borda

Você pode configurar a borda de um elemento, e configurar seu raio, sua forma, sua cor, sua sombra

Forma da borda

border-style: solid;

  • Tipos de formas para as bordas abaixo:

solid - Borda preenche todo o espaço

dashed - Borda pontilhada por linhas

dotted - Borda pontilhada por bolinhas

double - Linha dupla

groove - Borda tridimencional

Cor da borda

border-color: black;

Largura da Borda

border-width: 5px;

Bordas Arredondadas

  • O estilo abaixo configura todas as extremidades
border-radius: 10px;

Quanto maior o tamanho que você escolher maior será o arredondamento

  • Para configurar cada extremidade em específico com um arredondamento basta fazer como no código abaixo
border-radius: 0px 10px 0px 10px;

A ordem do que cada parâmetro muda é: cima esquerda, cima direita, baixo direita, baixo esquerda

  • Você pode configurar cada extemidade de uma vez em estilos diferentes
  • No exemplo abaixo configuramos o canto superior esquerdo
border-left-top-radius: 10px;

  • Você também pode usar porcentagem para definir o arredondamento

Com 50% de arredondamento e se a altura for igual a largura, fica um circulo perfeito

Borda Interna

A borda de um elemento, por padrão, fica por fora da box do elemento

Para fazer com que a borda seja dentro da box do elemento, configure o elemento da seguinte forma:

section#redes-sociais img {
    box-sizing: border-box;
}

Ao adicionar o parâmetro acima a borda começa a fazer parte do tamanho da box

Borda Externa

Quanto maior você escolher maior será o raio da borda

Você pode designar uma imagem para ser sua borda, ou seja, fazer uma borda personalizada

  • Você precisa de adicionar uma borda antes de adicionar a borda personalizada, caso contrário nem aparecerá a borda

  • Código para adicionar borda personalizada:
border-image-source: url('imagem.png');

  • Dividir a borda em vários pedaços:
border-image-slice: 20;

Não precisa ser em medida, basta colocar a quantidade de vezes que irá dividir, quanto maior o número mais divisões terá

  • Repetir da borda
border-image-repeat: repeat;

Contorno

Para configurar um contorno é da idêntica forma que se configura a borda, o que muda é que o parâmetro usado para configurar e o outline

  • Você pode fazer todas alterações que fazem com bordas, com os contornos
outline-color: #000000;

Sombras

Sombra da caixa

box-shadow: 1px 1px 1px black;
  • A forma mais simples de se implementar uma sombra e usando a shorthand box-shadow

Quanto a sombra vai se mover na horizontal - É definida pelo primeiro parâmetro da shorthand

Quanto a sombra vai se mover na vertical - É definida pelo segundo parâmetro da shorthand

Espalhamento da sombra - É definida pelo terceiro parâmetro da shorthand

Cor da sombra - É definida pelo quarto parâmetro da shorthand

Sombra para todos os lados

Se você quiser uma sombra para todos os lados, deixe o primeiro e o segundo parâmetro da shorthand vazios, e configure apenas o espalhamento (terceiro parâmetro)

Sombra interna - Para definir uma sombra interna usasse os parâmetros acima mas com o parâmetro inset antes

box-shadow: inset 1px 1px 1px black;
  • outset - É a configuração padrão (sombra para fora), utilizada da mesma forma que inset

Margem

Margem - É configurada pelo parâmetro margin e existem diversas formas de a configurá-la, cada direção em específico ou definir um raio para a margem

margin: 1px 1px 1px 1px;

  • Para centralizar um bloco dê o valor auto para margin
margin: auto;

Margem Automática - Ao criar um documento ele já vem automaticamente com uma margem para os elementos, essa margem é do elemento body, para a excluir basta colocar o valor da margem de body igual a 0

Acolchoamento

Acolchoamento - É configurada pelo parâmetro padding tem como configurar cada direção em específico ou definir um raio para todo o acolchoamento

padding: 1px 1px 1px 1px;

Altura e Largura

Altura - É configurada pelo parâmetro height, e existem diversas formas de atribuir valor a ela

height: 100px;

Largura - É configurada pelo parâmetro width, e existem diversas formas de atribuir valor a ela

width: 100px;

Altura Máxima - É configurada pelo parâmetro max-height, e existem diversas formas de atribuir valor a ela

max-height: 100px;

Largura Máxima - É configurada pelo parâmetro max-width, e existem diversas formas de atribuir valor a ela

max-width: 100px;

Altura Mínima - É configurada pelo parâmetro min-height, e existem diversas formas de atribuir valor a ela

min-height: 100px;

Largura Mínima - É configurada pelo parâmetro min-width, e existem diversas formas de atribuir valor a ela

min-width: 100px;

  • Os parâmetros de altura e largura, máximas e mínimas, servem para configurar o máximo e o mínimo de tamanho que um elemento poderá atingir

Apresentação

O comando “display” é usado para indicar de que forma um elemento irá se apresentar em um site

h1{
            display: ;
        }

block - O elemento se torna box-level

inline - O elemento se torna inline-level

inline-block - O elemento se torna um inline-block

none - O elemento se torna oculto (não aparece)

Posição

Absoluta - Ao tornar um elemento absoluto, ele ganha a propriedade de poder ocupar qualquer lugar da tela, independente do elemento ao qual ele pertence (no caso o elemento pai)

Outra coisa que irá acontecer é que ele não vai se limitar ao estilo que foi designado ao seu elemento pai, ele ficará independente, podendo ter seu próprio estilo

position: absolute;

Para o parâmetro absoluto permanecer dentro de seu elemento pai, seu elemento pai deverá ser relativo

Relativa - Se um elemento pai tiver a propriedade relativa, seus elementos filhos por mais que sejam absolutos, não poderão ficar fora de seu elemento pai

  • Os elementos absolutos continuam tendo todas propridades de um elemento absoluto, porém não podem ficar fora de seu elemento pai
position: relative;

Posicionar um elemento absoluto

As configurações para posicionar um elemento absoluto elas funcionam em relação ao elemento no qual o elemento absoluto pertence, se ele estiver dentro de um elemento relativo o posicionamento vai ocorrer dentro desse elemento, se ele não estiver dentro de nenhum elemento relativo o posicionamento vai ser em qualquer lugar do documento

Um elemento absoluto pode receber alguns parâmetros diferentes do que os de outro elemento, os parâmetros novos que usaremos para definir o posicionamento de um elemento absoluto são os parâmetros left, top, bottom e right

  • Left e right são os parâmetros que irão configurar o posicionamento vertical de um elemento
  • Top e bottom são os parâmetro que irão configurar o posicionamento horizontal de um elemento

Left - O quanto o parâmeto irá deslocar da esquerda para direita

left: 50%;

Top - O quanto o parâmeto irá deslocar de cima para baixo

top: 50%;

Right - O quanto o parâmeto irá deslocar da direita para esquerda

right: 10px;

Bottom - O quanto o parâmeto irá deslocar da baixo para cima

bottom: 10px;

  • Se você colocar os parâmetros left e top om o valor de 50% o ponto de início do parâmetro será exatamente no meio
  • Para o parâmetro absoluto permanecer dentro de outro esse outro deve ser relativo

Posicionar para direita ou esquerda

Existe um parâmetro que permite você escolher de que lado um elemento vai ficar em uma div

Exemplo você tem uma div principal, e dentro dessa div principal temos duas outras divs, e você quer deixar uma dessas divs para direita e a outra para esquerda

Para fazer isso você pode configurar o parâmetro float dentro do estilo dessas divs, confira abaixo:

div#principal > div#um {
    float: left;
}

div#principal > div#dois {
    float: right;
}
  • Ao fazer a configuração acima, uma div vai ficar posicionada a esquerda da div principal e a outra a direita da div principal

Transformações

  • O parâmetro “transform” serve para fazer várias alterações adicionais a um elemento, transformar ela de acordo como você queira
transform: ;

Mover o elemento

Usado para mover o elemento para outro local

transform: translate(-50%, -50%);

Translate é um parâmetro que deve receber dois valores, significado de cada valor:

  • Primeiro valor positivo, mexe para direita
  • Primeiro valor negativo, mexe para esquerda
  • Segundo valor positivo, mexe para baixo
  • Segundo valor negativo, mexe para cima

Se estiver em porcentagem, a quantidade que o elemento vai mexer vai variar de acordo com o tamanho do próprio elemento, um exemplo, se eu pedir para mexer 50% para o lado e o tamanho horizontal do elemento for de 500px, o elemento irá mexer 250px

Alteração no Texto

Usado para alterar a forma das letras de um texto, entre maiúsculas e minúsculas

text-transform: uppercase;

Além de minúsculas e minúsculas temos outras opções

  • Isto é usado para alterar o texto de alguma parte do seu documento de maneira mais simples

Barra de Rolagem para Blocos

  • Serve para adicionar barra de rolagem à blocos que tem conteúdos que ultrapassam o tamanho da tela

Isto torna responsiva um conteúdo que é muito largo e em aparelhos ou telas menores cria rolagem lateral no site, basta seguir os passos a seguir:

Configure a div usando o parâmetro a seguir:

div{
            overflow-x: auto;
        }

  • Overflow significa todo o conteúdo que ultrapassa a tela
  • Temos o parâmetro overflow, que configura os conteúdos que ultrapassam verticalmente e horizontalmente o tamanho da tela
  • Temos o parâmetro overflow-x, que configura os conteúdos que ultrapassam horizontalmente o tamanho da tela
  • Temos o parâmetro overflow-y, que configura os conteúdos que ultrapassam verticalmente o tamanho da tela
  • Ao o setar como auto, ele irá criar uma barra de rolagem dentro da div, se os conteúdos da div forem maiores que a tela

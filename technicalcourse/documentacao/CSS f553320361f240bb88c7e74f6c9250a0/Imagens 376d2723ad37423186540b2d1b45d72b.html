Imagens

Imagem

<img>

Ao digitar ”img“ faremos a abertura dessa tag:

<img src="" alt="">

src - será usado para colocar a imagem, com crtl + espaço

alt - será usado para escrever um texto que vai descrever a imagem

<img src="imagens/imagem.g.png" alt="Imagem Grande">

Tamanho da Imagem

Há diversas medidas que podem delimitar o tamanho de uma imagem, as principais são:

pixel (px) - Ao delimitar a imagem com pixels você indica quantos pixels da tela a imagem irá ocupar

Porcentagem (%) - Ao delimitar a imagem com porcentagem você indicar quantos porcento do local onde a imagem foi implementada ela poderá ocupar

Imagens Dinâmicas

Para fazer as imagens se adaptarem conforme o tamanho da resolução de cada aparelho usamos as imagens dinâmicas

<picture>

<picture></picture>

Dentro da tag <picture> você pode carregar uma imagem com a tag <img>

<picture>
        <img src="imagens/imagem.g.png" alt="Imagem Grande">
</picture>

<source:midia:type>

O parâmetro <source:midia:type> é utilizado para colocar uma regra, de que se o tamanho da tela for menor ou maior que a imagem, o site irá carregar outra imagem, assim as imagens do seu site se adaptarão dependendo de qual dispositivo ou resolução de tela o site está sendo executado

<source media="(max-width: )" srcset="" type="image/">

media=”(?-width: )“ - Você irá delimitar a partir de qual largura você irá abrir outra imagem

media=”(max-width: )“ - Se a resolução do aparelho for menor que do que a que você delimitou outra imagem será aberta

media=”(min-width: )“ - Se a resolução do aparelho for maior que do que a que você delimitou outra imagem será aberta

srcset - Dita qual a imagem vai ser aberta

type - Para designar o tipo da imagem que será aberta

<picture>
        <source media="(max-width: 1050px)" srcset="imagens/imagem.m.png" type="image/png">
        <img src="imagens/imagem.g.png" alt="Imagem Grande">
</picture>
  • Sempre é bom colocar 50px a mais do que é necessário para evitar que possa abrir por algum momento a imagem errada, ou a barra de rolagem ser acionada

<picture>
        <source media="(max-width: 750px)" srcset="imagens/imagem.p.png" type="image/png">
        <source media="(max-width: 1050px)" srcset="imagens/imagem.m.png" type="image/png">
        <img src="imagens/imagem.g.png" alt="Imagem Grande">
</picture>
  • A tag ”source“ sempre deve vir acima da imagem, quanto menor a imagem carregada mais acima deve estar
  • Funciona da mesma forma com ”(min-width: )“, só inverte a ordem de imagem pequena para grande de grande para pequena

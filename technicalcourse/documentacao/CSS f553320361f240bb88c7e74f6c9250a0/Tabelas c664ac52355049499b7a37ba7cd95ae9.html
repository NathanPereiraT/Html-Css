Tabelas

Retirar espaço entre células

Por padrão cada célula vem com um espaço entre elas, para retirar este espaçamento basta usar o parâmetro a seguir:

table{
            border-collapse: collapse;
        }
  • Por padrão o valor de border-collapse vem como separate (faz as bordas terem espaço entre si)

Largura/Altura da Tabela

Para tabela ficar mais larga/alta aumente a width/height da tabela, pois a largura/altura das colunas/linhas se adapta automaticamente para preencher toda a tabela

Distância do conteúdo para linha

Se um conteúdo de uma tabela estiver muito colado com as linhas o recomendado é amentar o padding do td

Alinhar conteúdo de uma célula

Para alinhar horizontalmente use o seguinte parâmetro:

td{
            text-align: center;
        }

Para alinhar verticalmente use o seguinte parâmetro:

td{
            vertical-align: middle;
        }
  • O uso do parâmetro vertical-align é recomendado só para tabelas

Zebrado

Para criar o efeito zebrado em sua tabela você deverá usar a seguinte pseudo-class:

tr:nth-child(2n){
            background-color: #777777;
        }

A pseudo-class se chama nth-child(), dentro do parênteses se coloca um valor, exemplo:

2n - Significa que a cada 2 elementos um deles receberá este estilo

2n-1 - Significa que a cada 2 elementos um deles receberá este estilo, porém a regra começa a ser aplicada um elemento antes (isso faz com que a ordem do zebrado se inverta)

Cabeçalho fixo

Para tornar o cabeçalho de uma tabela fixo primeiro teremos que trocar o position de table para relative, assim como a seguir:

table{
            position: relative;
        }

Após isso teremos que alterar o position do título(th) para sticky, assim como a seguir

thead > tr > th{
            position: sticky;
        }
  • Para indicar th é recomendado indica-lo assim como podemos ver acima (como elemento filho)

Depois teremos que configurar aonde da tabela o cabeçalho irá se fixar, assim como a seguir:

thead > tr > th{
            top: -1px;
        }
  • Ao colocar o valor de top como 0px temos um pequeno espaço que não é coberto encima, por isso, é recomendado colocar como -1 para que não fique com este espaçamento

  • Ao usar os comandos acima apenas o texto do cabeçalho se fixa ao topo da tabela, seu fundo não se fixa, então temos que colocar junto das configurações anteriores a cor de fundo do elemento

Personalização de Colunas

Temos um parâmetro que permite indicar uma coluna inteira da tabela de forma simples, e com esse parâmetro torna mais fácil personalizar uma coluna, já que por esse parâmetro indicar toda a coluna se personalizamos ele toda a coluna é personalizada, confira abaixo como se usa este parâmetro:

Primeiramente você deve colocar dentro de table o parâmetro colgroup

<colgroup>
            
</colgroup>

Dentro dele você indicará cada coluna usando o parâmetro col

<colgroup>
            <col>
            <col>
            <col>
            <col>
</colgroup>
  • O primeiro col é respectivo a primeira coluna da tabela, o segundo col à segunda coluna, o terceiro col à terceira coluna, e assim respectivamente

Com esses parâmetros que sozinhos indicam uma coluna inteira, torna mais fácil adicionar um estilo à coluna, pois basta alterar o estilos desses parâmetros, como podemos ver abaixo:

<col id="ColImportante">

No exemplo acima basta alterar o estilo do id que a coluna inteira seguirá este estilo

Se uma dessas colunas tiver span ela vai se referir a mais de uma coluna, como no exemplo abaixo, o parâmetro col vai referir a duas colunas (Exemplo: Se a segunda coluna tiver dois de span, vai se referir a segunda e terceira coluna, e com isso o próximo col que for colocado em colgroup vai se referir à quarta coluna)

<col span="2">
  • É mais recomendado colocar um col para cada coluna em colgroup e dar estilo separadamente para elas (sem usar span)

Tabelas Responsivas

Para tornar responsiva uma tabela que é muito larga e em aparelhos ou telas menores cria rolagem lateral no site, basta seguir os passos a seguir:

Primeiramente envelope toda sua tabela em uma div

Depois disso basta configurar essa div usando um parâmetro, como a seguir:

div#tab{
            overflow-x: auto;
        }

  • Overflow significa todo o conteúdo que ultrapassa a tela
  • Temos o parâmetro overflow, que configura os conteúdos que ultrapassam verticalmente e horizontalmente o tamanho da tela
  • Temos o parâmetro overflow-x, que configura os conteúdos que ultrapassam horizontalmente o tamanho da tela
  • Temos o parâmetro overflow-y, que configura os conteúdos que ultrapassam verticalmente o tamanho da tela
  • Ao o setar como auto, ele irá criar uma barra de rolagem dentro da div, se os conteúdos da div forem maiores que a tela
